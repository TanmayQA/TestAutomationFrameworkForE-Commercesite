# Test Automation Framework for E-Commerce Site

Scalable Java 17 Test Automation Framework built using Selenium WebDriver and TestNG.  
Supports parallel execution and LambdaTest cloud integration for cross-browser testing.

---

## Overview

This framework automates UI testing for web applications using:

- Java 17
- Selenium WebDriver
- TestNG
- Maven
- LambdaTest (Cloud Execution)

It follows Page Object Model design and supports data-driven testing using JSON.

---

## Features

- Java 17 based automation framework
- Selenium WebDriver UI automation
- TestNG for test management and parallel execution
- LambdaTest integration for cloud testing
- Data driven testing using JSON
- Environment configuration (Dev / QA / UAT)
- Page Object Model
- Maven build
- TestNG XML parameterization

---

## Prerequisites

Ensure the following are installed:

- Java 17
- Maven
- IntelliJ IDEA (or any Java IDE)
- LambdaTest account (optional for cloud runs)
